## 🖨️ Print to PDF - Best Settings / প্রিন্ট করার সেরা সেটিংস

### Method 1: Chrome/Edge (Recommended) ✅

1. **Open Invoice** in browser
   - Fill all information
   - Click **🖨️ Print Invoice** button
   - Or press `Ctrl + P`

2. **Print Settings / প্রিন্ট সেটিংস:**
   ```
   ✅ Destination: Save as PDF
   ✅ Paper size: A4
   ✅ Pages: All
   ✅ Layout: Portrait (লম্বালম্বি)
   ✅ Margins: Default
   ✅ Scale: 100% (or "Default")
   ✅ Options:
      ☑️ Background graphics (ENABLE THIS!)
      ☑️ Headers and footers (Optional)
   ```

3. **Important! অবশ্যই করুন:**
   - **"Background graphics"** অবশ্যই চেক করুন
   - এটি না করলে colors দেখা যাবে না
   - Scale 100% রাখুন

4. **Save** button click করুন

## 🔧 Troubleshooting PDF Issues / PDF সমস্যা সমাধান

### Issue 1: Colors Not Showing (সবচেয়ে Common) ⚠️
**Solution:**
```
Print Dialog → More Settings → Options
✅ ENABLE "Background graphics"
```
Without this, colors won't print! / এটা না করলে রঙ দেখাবে না!

### Issue 2: Layout Broken / লেআউট ভেঙ্গে গেছে
**Solution:**
```
- Set Scale to 100%
- Set Margins to "Default"
- Try different browser (Chrome recommended)
```

### Issue 3: Logo Too Large / লোগো খুব বড়
**Solution:**
```
Logo automatically resizes to max 160px in PDF
If still too large, resize your image file to 200x200px
```

### Issue 4: Text Cut Off / লেখা কাটা
**Solution:**
```
- Use Landscape orientation if content is too wide
- Or reduce Scale to 90%
```

### Issue 5: Multiple Pages / একাধিক পৃষ্ঠা
**Solution:**
```
This is normal if you have many items
Each invoice may be 1-2 pages
Page breaks optimized automatically
```

## 💡 Pro Tips / প্রো টিপস

### Tip 1: Test Print First
```
Before final PDF:
1. Do a test print
2. Check all sections
3. Verify colors show
4. Ensure nothing is cut off
```

### Tip 2: Logo Size
```
Recommended logo size:
- Width: 150-200px
- Height: 100-150px
- Format: JPG or PNG
- File size: < 100KB
```

### Tip 3: Save Template
```
1. Fill common information
2. Click "Save Draft"
3. Load for next invoice
4. Only change customer & items
```

### Tip 4: Consistent Look
```
Use same terms & conditions for all invoices
Save a template with your standard terms
```

### Tip 5: File Naming
```
Save PDFs with consistent naming:
- INV-20260306-0001.pdf
- ClientName-Invoice-Date.pdf
```

## 📱 Mobile Print / মোবাইলে প্রিন্ট

যদি mobile/tablet থেকে print করতে চান:

1. Open in Chrome Mobile
2. Menu (⋮) → Print
3. Select "Save as PDF" or your printer
4. Enable "Print backgrounds"
5. Save or Print

**Note:** মোবাইলে full design দেখানোর জন্য landscape mode use করুন।
