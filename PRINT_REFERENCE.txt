Problem: Colors not showing / রঙ দেখাচ্ছে না
Solution: Enable "Background graphics" ✅

Problem: Layout broken / লেআউট ভাঙা
Solution: Scale = 100%, Margins = Default ✅

Problem: Logo doesn't appear / লোগো দেখা যাচ্ছে না
Solution: Add talukder-ict.jpg to images/ folder ✅

Problem: Buttons showing in PDF / PDF তে বাটন দেখাচ্ছে
Solution: This shouldn't happen! Check print.css ⚠️

Before printing final PDF:
1. Fill ALL information completely
2. Review on screen (looks good = PDF will look good)
3. Press Ctrl+P for print preview
4. Check colors in preview
5. If preview looks good → Save PDF ✅
