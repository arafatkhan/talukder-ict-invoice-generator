## 🎯 Tips & Tricks / পরামর্শ

### ⌨️ Keyboard Shortcuts
- **Ctrl+P** → Print (প্রিন্ট)
- **Ctrl+S** → Save Draft (সংরক্ষণ)
- **Ctrl+L** → Load Draft (লোড)
- **Ctrl+N** → New/Clear (নতুন)
- **Ctrl++** → Add Row (সারি যোগ)

### 💡 Pro Tips
1. **Auto-save**: Draft saves automatically when you click Save
2. **History**: Last 10 invoices saved automatically
3. **Phone Format**: Auto-formats to +880 format
4. **Words**: Amount converts to Bengali & English words automatically
5. **Print**: Use Chrome/Edge for best results

### 🎨 Customization
- Edit company info in `index.html` (lines 45-53)
- Change colors in `css/style.css`
- Replace logo in `images/` folder

### 📱 Mobile Use
- ✅ Fully responsive
- ✅ Works on phones/tablets
- ✅ Same features as desktop

## ❓ Common Questions / সাধারণ প্রশ্ন

**Q: Do I need internet? / ইন্টারনেট লাগবে?**  
A: No! Works 100% offline / না! ১০০% অফলাইনে কাজ করে

**Q: Where is data saved? / ডেটা কোথায় সংরক্ষিত?**
